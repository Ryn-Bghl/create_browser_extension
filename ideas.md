## HOW TO BUILD AN EXTENSION

 [ ] create a popup.html # (Optional) Runs in web pages
 [ ] create a popup.js # (Optional) UI when clicking icon
 [ ] create a styles.css # JS for the popup
 [ ] create a icons folder # Extension icon images
 [ ] create a content.js # (Optional) styles for popup or injected pages
